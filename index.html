Hi Profile   //    July 2015

A web app that allows you to follow your favorite GitHub users. Hi Profile is built with React, React Router, Bootstrap, webpack, Axios, and the GitHub API. It is written with tomorrow in mind, using the excellent Babel JavaScript compiler and new features included in the ES2015 Language Specification.

Quiz Manager   //    June 2015

Todo apps may be the canonical exercise to learn a new JavaScript framework, but I prefer trying my hand at simple quiz applications. Quiz Manager provides a dashboard for managing the data to be consumed by such an application. It's built with AngularJS, Foundation, Node.js, and Express, and implements a RESTful API to allow CRUD operations on a collection of quizzes in JSON format.
